# CSS notes

- `#id`: selects the element with that `id`.
- `.class`: selects the element with that `class`.
- `<html-tag>`: selects all elements of that tag.
- `#main img`: The `<img>` tag inside the element with id `main`.

The CSS transition property is a shorthand property for:
- transition-property (Required)
- transition-duration (Required)
- transition-timing-function
- transition-delay

# GO notes

- `:=`: declare and assign a new variable
- `=`: assign an already declared variable
